____

echo:   Writes input text to output
____
    echo Julio      Writes "Julio" on std-output
    echo 'Julio'    Writes 'Julio' without interpreting on std-output

 Redirections:
  [command] > [file]: Change the output of command to the file. Overwrites!
  ==================
                     echo Julio > ReadMe.txt     Writes "Julio" and > redirects to ReadMe.txt
                     
                     
                     
  [command] < [file]: Change the input of command to the file
  ================== 
                      wc < ReadMe.txt       Reads the number of words from file
                      
                      
                      
  [command] >> [file]: Append the output of command to the file, add at the end
  ===================
                      echo Julio >> ReadMe.txt
                      
                      
  [command] | [command]: PIPE command. Takes output of left command as input of right command.
  =====================

ls:
__    List command    Lists the content of a folder   
  ls        Lists current folder/directory
  ls .      Lists current folder/directory
  ls ~      Lists My Home Directory
  ls Direct Lists the directory "Direct"
  
  ls -a     Show hidden directories and files which start with "."
  ls -l     Long listing: Access Permissions, last modified date...
  ls -t     Time sort by last modified
  ls -alt   The three previous ones
  ls -al
  ls -at
  ls -lt    
  ls -tl 
  ....      Should be obvious
  
  ls ReadMe.txt     Checks if ReadMe.txt exists /error message
  ls *.txt          Checks all files with that extension (.txt)
  ls Read*          Checks all files that start with Read

cd:     Change Directory
__
  cd Downloads        Will take you from /Users/user to /Users/user/Downloads , for example
  cd Downloads/       Will take you from /Users/user to /Users/user/Downloads , for example
  cd ..               Will take you up to father directory, back to /Users/user in the example
  cd                  Will take you to Home directory
  cd ~                Will take you to Home directory
  
  
mkdir:      Make Directory, build directory in current folder
_____
     mkdir Secrets    Will build the folder Secrets in current directory
     mkdir .secret    Makes a hidden directory: .secret
     mkdir Documents/Secret    Makes a Secret directory inside Documents
     
     
     
touch:      Make a file 
_____
     touch ReadMe.txt       Creates the file ReadMe.txt in the current directory


cp:     Copy 
__
   cp [file] [path]           Copies a file into that path
   cp ReadMe.txt .secret      Copies ReadMe into .secret, a hidden directory
   cp ReadMe.txt WriteMe.txt  Overwrites a destination file
   
   
mv:     Move!
__  
   mv [file] [destination]        Moves file to that path
   mv ReadMe.txt ../readme.txt    Moves ReadMe.txt to father directory
   mv ReadMe.txt Dir/             Moves ReadMe.txt to Dir
   mv Documents/*.txt Documents/Secret  Moves all txt files to Secret, in Documents
   mv Secrets/*txt                Moves all the txt files of Secrets to the current directory
   mv * ..                        Moves ALL files to parent directory

curl:     C- URL, See URL. 'Downloads' a webpage (sourcecode)
____
     curl 'http://google.com'                     Prints sourcecode of the page

grep:    Global Regular Expression Print
____    Returns the lines of file with certain data pattern (word)

     grep [OPTIONS] PATTERN [FILE...]
     grep shell 'dictionary.txt'            Returns all the lines with 'shell' at some place (seashell, shellac...)
     grep shell 'dictionary.txt' | less     Sends the lines to less to read easily
     curl -L 'http://google.com' | grep go  Searchs lines in Google.com for "go" 
     grep shell 'dictionary.txt' | wc       How many lines with shell in dictionary? (Word Count)
     grep -c shell 'dictionary.txt'         How many lines with shell in dictionary? (Count extention)
     grep --color shell 'dictionary.txt'    colors "shell" in the lines
     grep -n shell 'dictionary.txt'         line where found
     grep -i shell 'dictionary.txt'         ignore case sensitivity
     grep -w shell 'dictionary.txt'         lines WITHOUT pattern
